# Results

이 폴더는 공개용 성능 요약 파일을 포함합니다.

- `final_metrics_summary.csv`는 README에 표시된 최종 성능 지표를 CSV로 정리한 파일입니다.
- `metrics_by_video_anonymized.csv`는 원본 영상 식별자를 제거하고 `sample_id`로 대체한 영상별 지표 요약입니다.
- 원본 영상, 영상 제목, 채널명, URL, 원본 라벨, 로컬 경로는 포함하지 않습니다.
- 영상별 상세 결과는 식별정보 제거가 가능한 경우에만 익명화 형태로 제공합니다.
- 결과 수치는 재계산하지 않고 제공된 최종 평가 요약값과 원본 지표 파일의 숫자 컬럼을 문서화한 것입니다.
