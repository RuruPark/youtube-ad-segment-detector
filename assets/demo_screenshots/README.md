# Demo Screenshots

이 폴더는 demo viewer 화면 예시 또는 설명 이미지를 두는 위치입니다.

원본 유튜브 영상 화면은 저작권과 개인정보 가능성 때문에 포함하지 않습니다. 현재 포함된 이미지는 viewer 구조를 설명하기 위한 공개용 예시 이미지이며, 실제 영상 프레임이나 외부 이미지를 사용하지 않습니다.

- `demo_viewer_overview.svg`: local video placeholder, predicted ad intervals, skip button, timeline 구조를 보여주는 중립 도식 이미지
