# Error Analysis

## 진단 요약

Development Set 사후 진단 기준으로 patch4의 오탐 후보 수는 0으로 기록되었습니다. 남은 오류는 종료가 짧은 구간 3건, 후보는 있으나 선택되지 않은 구간 3건, 과확장 구간 1건입니다.

## 성능 지표 해석

성능 지표를 보면 광고 구간 포착률은 85.0%였지만, 예측 광고 정밀도는 67.8%였습니다. 이는 실제 광고를 놓치지 않기 위해 후보 구간을 넓게 잡는 과정에서 일부 비광고 구간이 함께 포함되었기 때문으로 해석할 수 있습니다.

평균 시작 오차는 38.4초, 평균 종료 오차는 43.4초로 남아 있습니다. 광고 구간의 존재를 찾는 것과 정확한 경계를 맞추는 것은 별도 문제이므로, 향후에는 종료 확장과 경계 축소 규칙을 더 정교하게 다듬을 필요가 있습니다. 영상 하나당 비광고 오탐 시간은 평균 55.8초였습니다.

## 오류 유형

| 유형 | 수 | 의미 |
| --- | ---: | --- |
| `prediction_partial_too_short` | 3 | 광고 시작은 잡았지만 종료가 짧음 |
| `candidate_exists_but_not_selected` | 3 | review 후보가 있으나 최종 탐지 결과로 선택되지 않음 |
| `overextended_prediction` | 1 | 실제 광고보다 길게 잡힌 탐지 결과 |
| `viewer_review_cases` | 7 | 사람이 경계를 확인해야 하는 잔여 케이스 |

## 개선 방향

- 종료가 짧은 구간은 guarded end extension 후보를 viewer에서 확인합니다.
- 협찬/제품명만 있는 구간은 오탐 guard를 유지합니다.
- 과확장 구간은 내부 기준점 기반 trim 또는 더 강한 종료 단서로 보정합니다.
- Test Set 평가는 규칙을 고정한 뒤 별도로 수행합니다.
