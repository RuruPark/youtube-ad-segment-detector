# Project Summary

## 문제 정의

긴 유튜브 영상에서 일반 콘텐츠 흐름과 분리되어 삽입되는 광고 블록의 시작과 종료 시점을 찾습니다. 목표는 광고/비광고 구간 분류가 아니라, 실제 스킵에 사용할 수 있는 광고 구간 타임스탬프를 만드는 것입니다.

## 접근 방식

1. OpenCV/FFmpeg, ResNet embedding, TransNetV2로 장면 전환 기준점을 생성합니다.
2. 기준점 주변 OCR에서 유료광고 고지, 협찬 표현, 제품명, 구매 유도 문구, 링크 안내 등을 찾습니다.
3. 오디오 feature는 같은 영상 내부 기준선 대비 변화량으로 해석합니다.
4. 규칙 기반 상태 전이가 광고 시작 후보, 광고 중 상태, 종료 후보를 추적합니다.
5. 후처리 단계에서 가까운 후보 연결, 강한 후보 승격, 종료 확장, 경계 축소, 오탐 guard를 적용합니다.
6. 정답 구간은 탐지 판단에 사용하지 않고, 결과 평가와 오류 분석에만 사용합니다.

## 평가 요약

제공된 최종 평가 요약 기준으로 광고 구간 포착률은 85.0%, 예측 광고 정밀도는 67.8%였습니다. 평균 시작 오차는 38.4초, 평균 종료 오차는 43.4초였으며, 영상 하나당 비광고 오탐 시간은 평균 55.8초로 나타났습니다.

공개용 성능 요약은 `results/final_metrics_summary.csv`에, 식별자를 제거한 영상별 지표 예시는 `results/metrics_by_video_anonymized.csv`에 정리했습니다.

## 저장소 성격

이 저장소는 프로젝트 구조와 탐지 설계를 보여주는 포트폴리오용 코드 정리본입니다. 원본 영상, private label, OCR 원문, review output은 포함하지 않았습니다. 별도 오픈소스 라이선스는 아직 지정하지 않았습니다.
